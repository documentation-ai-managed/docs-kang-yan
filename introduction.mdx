## Overview

Kang Yan revolutionizes restaurant operations by providing a seamless point meal ordering platform. You manage orders, customize menus, and serve customers faster with real-time updates and intuitive tools. Designed for efficiency, it reduces wait times and boosts satisfaction for both restaurants and diners.

<Callout kind="info">
Kang Yan supports unlimited tables and scales effortlessly from small cafes to large chains.
</Callout>

## Integrate with Your App

Embed Kang Yan orders using the JavaScript SDK.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  import { KangYan } from 'kang-yan-sdk';

  const client = new KangYan({ apiKey: 'YOUR_API_KEY' });
  const orders = await client.orders.list({ tableId: 'table-123' });
  console.log(orders);
  ```
  ```python
  from kang_yan import KangYan

  client = KangYan(api_key="YOUR_API_KEY")
  orders = client.orders.list(table_id="table-123")
  print(orders)
  ```
</CodeGroup>
